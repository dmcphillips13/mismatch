# Mismatch

NHL moneyline discrepancy engine scaffold.

## Quick start

```bash
pnpm install
pnpm dev
```

## Environment

Copy `.env.example` to `.env.local` and fill in values.

## Current scope

- Minimal Next.js app shell
- Project dependencies for data pipeline, retrieval, and agent tooling
- Ready for implementation of `scripts/prepare_summaries.ts` and follow-on modules
